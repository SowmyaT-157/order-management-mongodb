## Installation & setup

1. To clone the project enter the below command in the terminal with the path you want to create your project folder.

#### Clone:

- https://github.com/SowmyaT-157/order-management-mongodb

- To Install Node.js click the below link
       [Node.js](https://nodejs.org/)
- To check the Node.js
    - node -v
    - It will display the version of the node
- Navigate to project directory.
  
    - `cd` project path
- Install the dependencies using below command.
 ``` bash
      npm install
 ```
1. To start the project use the below command in the terminal.
* To run the application, first build the application by using given command.
```bash
    npm run build
```
* If you can run the application use below command
```bash
    npm run start
  ```
